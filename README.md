# Shikhar Lab Website

https://igc-lab-kut.github.io/shikharlab-hp/

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `index.html` | トップページ |
| `members.html` | メンバー一覧 |
| `publications.html` | 論文一覧 |
| `projects.html` | プロジェクト一覧 |
| `style.css` | 共通スタイル |

## 編集方法

- **メンバー追加**: `members.html` の該当セクションに `member-card` を追加
- **論文追加**: `publications.html` の該当年に `publication-item` を追加
- **プロジェクト追加**: `projects.html` に `project-card` を追加
- **ニュース追加**: `index.html` の `news` セクションに `<li>` を追加

## デプロイ

GitHub Pages で自動公開（`main` ブランチにプッシュで反映）
